Dollar Hegemony — BRICS vs G7 Analysis

End-to-end ML research system quantifying US Dollar dominance impact on BRICS economies. Built a novel Dollar Stress Index (DSI) from scratch. Three-model pipeline — XGBoost with SHAP interpretability, LSTM for sequential forecasting, and a multi-country Transformer with cross-country attention. Blended via Ridge meta-ensemble. Live interactive dashboard with 7 tabs and FastAPI backend.

PythonPyTorchXGBoostPlotly DashFastAPIDocker
AI / ML ↗ GitHub ↗ Live Dashboard

India Crime Analytics Dashboard

Crime intelligence platform built on NCRB data — 755,380+ crimes tracked across 28 states + UTs spanning 7 years. Features 11 analytics modules: interactive SVG India map color-coded by crime risk, trend analysis, state report cards, and side-by-side state comparisons. Includes Groq AI chatbot for natural-language crime queries and an emergency helplines directory.

Multilingual AI fitness companion supporting 4 languages (EN / ES / JA / KO) with auto-detected units (kg/lbs) by country. Tracks workouts, logs nutrition with macro calculator, and delivers personalized coaching via Groq AI chatbot and food scanner. Full authentication with JWT + Google OAuth.
